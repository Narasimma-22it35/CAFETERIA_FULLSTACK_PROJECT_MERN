# 🍽️ Smart Campus Cafeteria - Fullstack MERN Project

A premium, modern cafeteria management system designed for campus environments. This project provides a seamless ordering experience for students and a powerful management dashboard for administrators.

## ✨ Key Features

### 🎓 For Students
- **Modern Auth**: Immersive glassmorphism login/register experience.
- **Visual Menu**: Digital menu featuring high-quality AI-generated food visuals.
- **Smart Checkout**: Integrated simulated payment gateway (UPI, Card, Wallets).
- **Digital Invoices**: Instant receipt generation with print-ready layouts.
- **Feedback System**: Share your dining experience directly with the management.

### ⚡ For Administrators
- **Dashboard**: Real-time overview of orders and sales.
- **Order Management**: Track and update order statuses from "Placed" to "Delivered".
- **Feedback Review**: Monitor student satisfaction and improve services.
- **Menu Management**: Control dish availability and details.

## 🛠️ Tech Stack
- **Frontend**: React.js, Framer Motion, Axios, React Hot Toast.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).
- **Styling**: Vanilla CSS with modern Glassmorphism & Responsive Design.

## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/Narasimma-22it35/CAFETERIA_FULLSTACK_PROJECT_MERN.git
cd CAFETERIA_FULLSTACK_PROJECT_MERN
```

### 2. Setup Backend
```bash
cd backend
npm install
# Create a .env file with MONGO_URI and PORT
npm start
```

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```

## 📸 Preview
Open your browser at `http://localhost:5173` to explore the premium campus dining experience!
